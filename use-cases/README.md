# System Design Use Cases

Real-world system design problems and their solutions, organized by domain and complexity level.

## Categories

### Social Media
Designing platforms for social interaction, content sharing, and real-time communication.
- Examples: Twitter, Instagram, Facebook, Discord

### E-Commerce
Building systems for online shopping, inventory, and transactions.
- Examples: Amazon, eBay, Shopify

### Real-Time Systems
Applications requiring instant data updates and low-latency responses.
- Examples: Live streaming, real-time collaboration, gaming

### Data Processing
Systems for large-scale data ingestion, processing, and analytics.
- Examples: Data warehouses, analytics platforms, log processing

---

Each use-case directory contains:
- `design.md` - High-level design overview
- `requirements.md` - Functional and non-functional requirements
- `architecture.md` - Detailed system architecture
- Additional specialized documents as needed
